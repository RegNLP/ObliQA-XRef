ObliQA-XRef Human Audit — Instructions

                

                
            

            

                
Goal: Audit dual-passage QA items for cross-reference relevance. Make all judgments
                    using only the displayed passages.

                
Scope: You will see a Question, a Source passage, and a Target passage. In some
                    items, an ID or a short “Reference Text” may appear near a passage; treat these as hints only—decide
                    based on the passages.


                
1) Question quality

                
    
                    
  • Yes: the question is clear and understandable.
    • 
                    
  • No: wording is unclear, ambiguous, or ungrammatical.
    • 
                


                
2) Passage relevance to the question

                
    
                    
  • both: both passages are needed or clearly related.
    • 
                    
  • only source: only the Source passage is related.
    • 
                    
  • only target: only the Target passage is related.
    • 
                    
  • neither / mismatch: neither passage supports the question.
    • 
                


                
3) Explicit citation from Source → Target

                
    
                    
  • Yes: the Source explicitly refers to the Target (e.g., section/paragraph
                        number, title/identifier, direct cross-reference).
    • 
                    
  • No: no explicit reference; topical overlap or paraphrase does not count.
    • 
                


                
Rules: Do not use outside knowledge. Keep comments brief (1–2 short
                    clauses), e.g., “only S related”, “no explicit citation”, “question unclear”. Use Save to download
                    progress.
